# Win98 怀旧屏保系列

来自 知乎, [Seahorse](https://www.zhihu.com/question/31919895/answer/101201842), windows98 版本的屏保？

## 怀旧屏保使用说明

> Win10 系统下，将所有 SCR 文件和 DLL 文件解压至 `C:\Windows\`，右键点击任意 SCR 文件安装即可使用。
>
> DLL 屏保系列是 Win98 系统自带屏保，由每个 SCR 文件和同名 DLL 文件组成，同时需要`wildlb32.dll`和`wl32dll.dll`两文件支持才能够使用。
>
> 在 Win10 系统中，所有 DLL 屏保都统一显示为一个名为 **“Wildlife 32”** 的屏保，取按字母顺序排列最靠前的一个。
>
> 所以只要把你想要使用的 DLL 屏保的 SCR 文件和对应的 DLL 文件改名为`001.scr`和`001.dll`即可使用该屏保。
>
> 比如如果你想使用神秘之物屏保，只需要把`Mystery.scr`和`Mystery.dll`分别重命名为`001.scr`和`001.dll`，默认屏保选择“Wildlife 32”就可以了。
